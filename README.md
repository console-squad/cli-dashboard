**Running the demo**

    npm install

    // dev mode
    npm run dev:dt
    npm run dev:covid

    // prod mode
    npm run start:dt
    npm run start:covid

Works on Linux, OS X and Windows. For Windows follow the [pre requisites](http://webservices20.blogspot.co.il/2015/04/running-terminal-dashboards-on-windows.html).
